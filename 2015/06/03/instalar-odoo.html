Instalar Odoo y no moriri en el intento

Date mié 03 junio 2015 Tags odoo / installar / postgresql

Esta es la primera entrada que escribo en este blog, ya que el blog se basara en gran parte python y en Odoo, decidi empezar con un post en el que se explique como instalar odoo.

Las indicaciones que aqui dare sera para Debian por lo que es muy probable que funcione en cualquiera de sus derivados.

En la web Nightly Odoo podremos encontrar los compilados, y los repositorios para Debian y Red Hat, y los .exe par Windows.

Bueno lo primero que debemos hacer es instalar es nuestro servidor de base de datos PostgreSQL, luego de esto podremos instalar odoo.

Instalando PostgreSQL

En una terminal como root ejecutamos los siguientes comandos:

#apt-get update
#apt-get upgrade
#apt get install postgresql

Esperamos que termine este proceso, en el que actualizamos el sistema y luego instalamos PosgreSQL.

Instalando Odoo

Lo primero que debemos hacer es ingresar el repositorio de odoo a nuestros repositorios para eso debemos editar el archivo /etc/apt/sources.list con nuestro editor favorito, y agregar el siguiente repositorio deb http://nightly.odoo.com/8.0/nightly/deb/ ./

#nano /etc/apt/sources.list

Ahora despues de haber hecho la modificacion procedemos a instalar odoo

#apt-get update
#apt-get install odoo

Al finalizar la instalacion ya tendremos Odoo listo para ser usado, para comprobar esto entramos en nuestro explorador y vamos a la siguiente direccion:

http://localhost:8069

Con esto despido el primero de muchos post relacionados con Odoo.
