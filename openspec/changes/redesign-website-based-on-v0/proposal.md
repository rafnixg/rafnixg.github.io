# Change: Redesign Website Based on v0 Version

## Why
The current portfolio website has a functional design but could benefit from a more modern, professional layout inspired by the v0-rafnixg-dev.vercel.app version. This redesign will improve user experience, better showcase projects and articles, and align with current web design trends.

## What Changes
- Restructure the homepage layout to match the v0 design: hero section, specialties, projects grid, articles section, contact footer
- Update HTML structure for better semantic markup and sections
- Redesign CSS for modern styling, improved typography, and responsive design
- Add projects section with cards displaying key projects
- Add articles section showcasing recent blog posts
- Update navigation and contact information
- Ensure dark theme compatibility and mobile responsiveness

## Impact
- Affected specs: website capability (layout and content structure)
- Affected code: index.html (major restructure), style.css (redesign), script.js (potential updates for new interactions)
- No breaking changes to external links or functionality, but visual overhaul
