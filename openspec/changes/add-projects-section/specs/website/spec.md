## ADDED Requirements
### Requirement: Projects Section Display
The website SHALL display a projects section showcasing completed work.

#### Scenario: Projects section visible
- **WHEN** a visitor loads the homepage
- **THEN** a projects section is displayed below the skills section
- **AND** each project shows title, description, technologies, and links

#### Scenario: Responsive project cards
- **WHEN** viewed on mobile devices
- **THEN** project cards stack vertically and remain readable
