# Change: Add Projects Section

## Why
The portfolio website currently showcases skills and contact information, but lacks a dedicated section to highlight completed projects and work samples. Adding a projects section will provide visitors with concrete examples of the developer's capabilities and achievements.

## What Changes
- Add a new "Projects" section to the main page
- Include project cards with title, description, technologies used, and links
- Ensure responsive design for the new section
- Update navigation to include projects section

## Impact
- Affected specs: website capability (new section)
- Affected code: index.html, style.css, script.js (if needed for interactivity)
