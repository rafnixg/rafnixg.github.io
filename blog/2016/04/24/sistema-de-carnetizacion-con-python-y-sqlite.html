Sistema de Carnetización con python y SQLite

Publicado el dom 24 abril 2016 en python

Me he decidio a publicar un codigo que todavia esta muy verde(Beta), pero creo que con un poco mas de tiempo podre tener un buen sistema, he programado un pequeño sistema para generar las imagenes (Carnets) necesarias para ser impresas en PVC, inicialmente esta diseñado para ser usado por una Aldea de la Misión Sucre.

Este sistema guarda la informacion necesaria para generar las imagenes para los Carnets, a travez de una interfaz grafica hecha con PyQT4, el codigo detras de este sistema esta hecho en Python, y la base de datos esta creada en SQLite, lo hace portable al sistema.

Para ver el codigo ir a mi cuenta GitHub Rafnixg.
